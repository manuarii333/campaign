🚀 Guide de mise en place
1
Ajouter le secret MERCHANT_ENCRYPTION_KEY dans Cloudflare Dashboard → Workers → payzen-hcs → Settings → Variables and Secrets (64 caractères hex — générer avec openssl rand -hex 32)
2
Cliquer sur ⚙️ Migration DB ci-dessus pour créer la table merchants en base D1
3
Cliquer sur + Nouveau marchand et renseigner les clés PayZen du marchand — elles sont chiffrées AES-256 avant stockage
4
Copier le Merchant ID généré (ex: mrc_abc123) et l'intégrer dans le widget HTML du marchand avec data-merchant-id
5
Le marchand n'a jamais accès aux clés secrètes — seul le Worker y accède via D1 chiffré
