Présentation

Andromeda Campaign Builder est une application web autonome développée par HCS Tahiti pour créer, éditer et exporter des campagnes publicitaires Facebook Ads avec paiement PayZen intégré.

L'application tient dans un seul fichier HTML — aucune installation, aucun serveur requis pour l'usage basique. Elle peut être ouverte directement dans un navigateur ou déployée sur un serveur web.
